About The Project

This project implements a quantum-classical hybrid solution for the Vehicle Routing Problem (VRP). By leveraging the Quantum Approximate Optimization Algorithm (QAOA), we explore the potential of quantum computing in logistics optimization.

01. Clustering

Large datasets are partitioned using K-Means clustering to create manageable quantum sub-problems.

02. QUBO Formulation

Routing constraints are mapped to a Quadratic Unconstrained Binary Optimization (QUBO) model.

03. Quantum Optimization

StatevectorSampler and COBYLA optimizer execute the QAOA circuit to find optimal routes.
